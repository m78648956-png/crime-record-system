## 🔍 About

**Crime Record Management System (CRMS)** is a full-stack web application designed to digitize and streamline crime record management for law enforcement agencies. It provides a public-facing website for citizens to track FIRs and file complaints, along with a secure admin/officer dashboard for managing crime records, criminal profiles, officer assignments, and generating analytical reports.

### 🎯 Purpose
- Replace traditional paper-based crime record systems
- Provide transparency through public FIR tracking
- Enable citizens to file complaints online
- Offer real-time crime analytics and reporting
- Ensure role-based secure access for different user levels

## ✨ Features

### 🌍 Public Website
| Feature | Description |
|---------|-------------|
| 🏠 Landing Page | Professional homepage with animated particles, stats, and service information |
| 🔍 FIR Tracking | Citizens can track their FIR status using FIR number |
| 📝 Online Complaints | Public complaint submission form with urgency levels |
| 📞 Contact Info | Emergency helpline numbers and contact details |
| 📱 Responsive | Fully responsive design for all devices |

### 🔐 Authentication & Security
| Feature | Description |
|---------|-------------|
| 🔑 Secure Login | Encrypted login system with password toggle |
| 👥 Role-Based Access | Separate access levels for Admin and Officers |
| 🔒 Session Management | Secure session handling with auto-logout |
| 👁️ Password Visibility | Toggle password visibility on login |

### 👨‍💼 Admin Dashboard
| Feature | Description |
|---------|-------------|
| 📊 Analytics Dashboard | Real-time statistics cards, crime charts, and activity feed |
| 📋 FIR Management | Complete CRUD operations for crime records |
| 👤 Criminal Profiling | Add/edit/delete criminal profiles with status tracking |
| 👮 Officer Management | Manage officers, assign ranks, and create login credentials |
| 📢 Complaint Management | View, review, and convert public complaints to FIRs |
| 📈 Report Generation | Summary, monthly, and crime-type analysis reports |
| 📋 Activity Log | Track all system actions with timestamps |
| 🔔 Notifications | Alert system for critical cases and pending complaints |

### 👮 Officer Portal
| Feature | Description |
|---------|-------------|
| 📝 Register FIR | Multi-step form with case info, people, and details |
| 📂 View Records | Browse, filter, and search all crime records |
| 🔍 Advanced Search | Search across cases and criminal records |
| 📄 Reports | Generate and print various reports |

| 💾 Local Storage | Persistent data storage in browser |
